## Install 

- install python
- $ pip install jupyter
- $ git clone https://github.com/pedrodiamel/nettutorial.git nettutorial
- $ cd nettutorial
- $ jupyter notebook
- open your host browser and paste address given by jupyter
- click on ibackprop.ipynb
